First, write the series (the infinite

It works nicely.

Because this theorem is true,

(whether it represents a number)

How do we capitalize on this?
There are many theorems we're
going to learn about series
(about convergence of series), and
many of them have this general form.
(This is sort of a template.)
Some condition implies that
the series is convergent,

if for all natural numbers n,
some property...
(I am writing in red here.
(This is generic.))
If for all natural numbers n,

Well, because of
what I just explained

if we have proven a theorem like this,
then we immediately have
a more general theorem,
and instead of saying for all
natural numbers n,
I can obtain the same conclusion
if the red property is only true
for all LARGE VALUES of n.
The precise definition of
"for all large values of n"
is "for all values
of n starting at one".
Or more formally,
if there exists a natural number n₀, such
that the property in red is true for n₀,
and for all natural numbers afterwards.
This is just a formal way to
say "for all large values of n".
In summary, whenever we get a
theorem like the first one,
which will probably be simpler to
write and state and prove,

theorem, like the second one here,
which is more general,
(and is going to be more applicable
and more helpful in examples)
and we don't have
to prove it explicitly!
This will come in handy,
and this is why we like to talk
about the TAIL of a series.
When it comes to the
convergence of a series,
the only thing that matters is what
happens for large values of n.
