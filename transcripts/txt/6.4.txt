An optimization problem is an application
of calculus to a physical situation
where I want to make a certain quantity as
large as possible or as small as possible.
Here is an example.
A woman gets on her boat at point A on
one side of a straight river, 3km wide.
She wants to get to point B,
8km downstream on the opposite side.
She could row the boat directly across the
river and then run for 8km,
or she could row directly from A to B,
or she could row
to some intermediate point and then run.
If she can row 6km/h and run 8km/h,
where should she land 
to reach B as soon as possible?
At this moment I invite you to pause the
video and try to solve the problem.
Even if you cannot fully solve it, if you
spend some time trying to get started,
setting it up, drawing a picture, and
understanding what it means,
my explanation will be much more useful.
The first part of the problem
is the modelling.
We need to transform the word problem
into a mathematical precise question.
In this case the question will be
something like:
I want to find the maximum or the minimum
of this specific function
on this specific domain.
Let's draw a picture.
We have a straight river,
and the woman begins on one side of the
river, let's call this point A.
We also know
the width of the river is 3km,
and she wants to row to some point
on the other side, let's call it B,
and we know B is 8km downstream from A.
Now as the problem says, she could row
straight and then run,
or row directly to B, or most likely,
to some intermediate point.
So the path she is going to take is
probably rowing somewhere there
and then running all the way to B.
The other information we have is that
she rows at 6km/h and she runs at 8km/h.
And what we want to figure out is
the position of this point here.
So let me use a variable for it.
I could do it in various ways;
I could for example use this angle
as the variable.
I think it's going to be simpler
to call x this distance,
and that's what I want to figure out,
what the value of x is.
Now we need an equation
for the total time,
because that's what
I'm trying to minimize,
the total time she spends 
moving from A to B.
And I can say that the total time
is going to be broken into
the two parts of the trip.
Let me give a name to this point:
let's say we call it C.
The total time is the time from A to C
plus the time from C to B.
And now what I need is to write
an explicit equation for each one of them.
For that I will need to know the value
of the two distances she runs and rows.
The distance from C to B is 8 - x.
As for the distance from A to C, 
I have a right triangle here:
this side is 3, this side is x,
so using Pythagoras' theorem,
I know that this length is √(x²+9), and
I think now I have all the data I need.
I can write the total time as follows:
first I need the time from A to C.
Well the distance is √(x²+9),
and the velocity for rowing is 6km/h,
therefore the time will be the distance,
√(x²+9), over the velocity, 6.
And then to go from C to B,
the time from C to B is going to be 
the distance, 8 - x,
divided by the velocity for running, which is 8 km/h,
and that's the total time.
I should specify the domain.
I cannot forget that.
So the values of x
have to be between 0 and 8.
It wouldn't make sense to row to a point
here to the left or to a point past B,
so it's going to be between 0 and 8.
And I definitely need to
include the endpoints.
I'm looking for
the minimum of this function,
and it could very well happen at 0,
or it could very well happen at 8,
maybe those are the optimal ways to go.
And now I have it: this is the function
I want to minimize in this domain.
That completes the setup of the problem.
The second part of the problem is easier:
we just need to use calculus
to find the minimum of this function.
Before we jump happily
into taking derivatives,
I'd like to make an observation.
This function is continuous,
and the domain is a closed and bounded
interval, it includes both endpoints.
Therefore I know, 
by using the extreme value theorem,
that this function must have a minimum.
And I know more:
the minimum will either be a critical
point or it will be one of the endpoints.
Therefore all I have to do is find the
critical points, find the endpoints,
and compare them, and I'll be done.
I don't need to do any more justification.
But that is only because I could use
the extreme value theorem,
because the function was continuous and
the domain included both endpoints.
Well I know what the endpoints are,
so let's find the critical points.
For that I need to figure out
what the derivative is.
And now I need to figure out
when the derivative is 0.
So I will escape the details,
do the algebra at your own pace,
but after doing a bit of calculation,
this function ends up having
one single critical point:
the derivative is 0 exactly when
x is 9/√7 km.
Now this is the only critical point
inside the domain.
There could be others outside the domain,
but I don't care;
I only care about what happens
inside the domain.
So I have my critical point,
and I also have my two endpoints,
so let's compare them.
And as a comment, yes,
we definitely need to use the endpoints.
It's possible the minimum is there.
For example,
if the woman could row
much faster than she could run,
then definitely the optimal path would be
to go directly from A to B,
and the minimum would happen when x = 8.
And if the woman could run much faster
than she could row,
then definitely the optimal path would be
to go from A straight here
and then run all the way to B,
and the minimum would happen at x = 0.
And I don't know what much faster
means in this case,
so it is possible the minimum is
at an endpoint, and either way,
in order to use extreme value theorem I
definitely need to include the endpoints.
So let's compare the three points,
the three candidate points.

When x = 0, the time is 1.5h.
When x equals 9/√7,

the time is 1 + (√7)/8 h,
which is approximately 1.33h.
And when x = 8km, the time is (√73)/6 h,

And of all those three, the smallest one

Therefore after all, the critical point

The minimum of this function
occurs when x = 9/√7km,
and that's the answer to the problem.
