In this video,
I will write a proof of the power rule
when the exponent is a positive integer.
The power rule states that the derivative
of xᶜ with respect to x is cxᶜ⁻¹,
where c is a constant. This identity is
true for any real number c.
In a later video which I will link in the
description,
I will write a proof that works for all
real numbers c,
but it requires heavy machinery.
There are some easier proofs that work
for smaller sets of exponents.
Specifically, in this video,
I will present a proof that works when
c is a positive integer.
This proof is by induction on the
exponent c.
There are also direct proofs in case you
prefer those.
Like all proofs by induction, I begin
with the base case. Here it is c = 1.
I need to show the identity is true when
c = 1,
so I want to show that the derivative of x
with respect to x equals 1·x⁰,
or simply 1.
I'll define the function f by f(x) = x.
Then, by definition of derivative,
f'(x) = lim_(h → 0) [f(x + h) - f(x)] ÷ h.
Then I substitute the value of f,
and everything simplifies,
and it simplifies a bit more,
and the limit is 1.
That's it. The base case is often simpler.
Let's move to the induction step,
which is often more interesting.
I fix a positive integer c.
I assume the identity is true
for this value of c,
and I want to prove it is also true
for c + 1.
I can write xᶜ⁺¹ as xᶜ·x.
This is useful, because it relates the 
function
whose derivative I want to compute,
to the function whose derivative I know.
These two functions are equal,
so the derivatives are equal as well.
Now I have a derivative of a product.
I can use the product rule.
I get the derivative of xᶜ times x,
plus xᶜ times the derivative of x.
I can write the derivative of xᶜ using
my assumption, the induction hypothesis.
And I know the derivative of x is 1.
This was the base case.
Now let's group these terms together.
And I get exactly what I needed.
This ends the induction step.
I have proven the identity is
true for c = 1,
and I have proven that if the identity
is true for a value of c,
then it must also be true for
the next value of c.
So we are done.
