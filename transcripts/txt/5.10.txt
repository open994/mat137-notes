In this video, I want to explain
why integration is possible.
This may sound like a weird topic,
but it actually fits very well into
this sequence of videos,
because it is our second application
of the Mean Value Theorem.
I will explain what I mean
with an example.
Have a look at this exercise.
I want to find all functions f
whose derivatives are x^2.
So, a function defined for all real numbers
whose derivative is always x^2.
By guessing, by trial and error,
you can get one answer.
This is (1/3)x^3.
It's easy to see that it works
once we think about it.
But, it is not the only solution.
I can also take (1/3)x^3 + any constant,
and that will also be a solution,
since adding a constant does not
change the derivative.
But, could there be any other solutions?
Are those all the functions
that satisfy this, or are there more?
In principle, if we think we are done,
this is something we have to prove.
Perhaps there are other functions
we haven't thought of.
The answer is NO: there are no others.
But, we do have to prove it.
Specifically, I want to prove this claim.
I want to show that IF a function
satisfies this equation
(if the derivative is x^2)
THEN there must exist a constant C
such that the function is (1/3)x^3 + C.
That will guarantee that we are done.
Fortunately, the proof is short
with the things we know so far.
It goes in three steps.
First, a theorem we already know.
We know that IF a function has zero
derivative on an open interval,
THEN it must be constant.
That is something we proved
in the previous video,
and we are going to be using it here.
This was an application of the
Mean Value Theorem.
Second, a corollary.
A "corollary" is a theorem that is short,
and is a quick consequence
of another theorem.
So, a quick consequence of
the previous theorem.
If, instead of one function I begin
with two functions f and g,
and I assume they have the same
derivative, on an open interval,
then what can I conclude?
Well, if they have the same
derivative, I can conclude
that their difference must be constant.
That's because the difference will
have zero derivative.
So, by the theorem,
the difference must be constant.
How does this help me?
Well, in the third step
I can now prove the claim.
If I begin with a function f and I
assume that its derivative is x^2,
then the derivative of f is the same
as the derivative of the function (1/3)x^3.
By the corollary, the two functions
may differ by a constant,
and that's exactly what I was
trying to prove.
So, I was able to prove it,
and it turns out it is true.
I had all the solutions.
This bears repeating, because
it is very important.
Without the Mean Value Theorem,
I would have only been able to say
that any function of the form (1/3)x^3 + C
is a solution to this equation,
and that these are some of the solutions.
But, thanks to the Mean Value Theorem,
I am able to say that these
are ALL the solutions.
This is much more important.
And of course, you can notice that there
is nothing special about this function.
The same could apply
to any other function.
In fact, this is at the core of
all integration methods.
Integration methods are techniques
to try to find all functions
with a given derivative.
They are all based on this same idea.
You just find one, and you add "+ C."
But, we needed to know that
that actually works;
that it will produce ALL the solutions.
So, the Mean Value Theorem is at the
foundation of all integration theorems.
And therefore, I think that this
application by itself
justifies all the work that
we had to put
into learning how to prove
the Mean Value Theorem.
