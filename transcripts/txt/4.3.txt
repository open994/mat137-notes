In this video, I will study various
examples of pairs of inverse functions.
For the theory behind inverse functions,
see the previous video.
I will begin with a very simple function:
the function f, defined by the equation
f(x) = 2x + 1.
The graph of f is a line.
The domain is all of the reals,
and the range is all of the reals.
Everything should work fine
for this example.
The inverse is defined by this statement:
x is f⁻¹(y) IF AND ONLY IF
y is f(x).
We have an equation for f:
f(x) = 2x + 1.
So I can write y = 2x + 1,
and then I have to solve for x.
In this case it is easy:
we get x = (y-1)/2.
So I have an equation for the
inverse function: f⁻¹(y) = (y-1)/2.
If I want to do the same
for a different function,
the process will always be the same;
however, solving for x
may be more difficult!
Sometimes it is even impossible
to solve explicitly.
Since the calculations are easy
with this particular example,
let’s use it to verify the identities...
...that we know pairs of
inverse functions MUST satisfy.
We know that if I compose a function and
its inverse, I should get the identity.
Let’s check it!
If I write f⁻¹(f(x)):
first I use the equation for f⁻¹.
I get (f(x) - 1)/2.
Then I use the equation for f.
I get ((2x + 1) - 1)/2.
And yes, this simplifies to x.
Now let’s compose the other way around!
If I write f(f⁻¹(y)):
first I use the equation for f.
I get 2(f⁻¹(y)) + 1.
Then I use the equation for f⁻¹.
I get 2((y-1)/2) + 1.
And yes, this is y,
as it should be.
For my second example,
let’s look at a classic:
the exponential function
and the natural logarithm function.
This is not an example
of COMPUTING an inverse;
but rather, of how to DEFINE
a new function as an inverse.
The exponential and logarithm
ARE inverses of each other.
They satisfy the definition
of a pair of inverse functions.
More than that,
this statement can be used to DEFINE one
of them, if we already know the other.
You probably did this in high school.
You took the exponential
as a starting point,
assumed you already knew what
an exponential was, and its properties,
and then you used THIS STATEMENT
to DEFINE the logarithm.
But it can also be done backwards!
In analysis, when we try to define
everything rigorously from scratch,
it is common to START with the logarithm.
The logarithm can be constructed
as an integral without assuming anything,
and then we use THIS PROPERTY
to define exponential instead.
Let’s look at their graphs.
We see that the domain of the exponential
is all of the reals,
which is the range of the logarithm.
And the domain of the logarithm
is the positive reals,
which is the range of the exponential,
as it should be.
Notice that one graph is
the REFLECTION of the other
with respect to the diagonal
of the first and third quadrants.
This always happens with
pairs of inverse functions.
Switching from a function to its inverse,
means swapping the roles of x and y.
Now think of swapping the x axis
and the y axis:
it is the same as reflecting
with respect to the diagonal.
That’s why graphs of pairs of inverse
functions always have this property.
Finally, the composition of a function
and its inverse should be the identity.
Indeed,
ln(eˣ) is x,
FOR ALL real numbers x.
And eˡⁿ⁽ʸ⁾ is y,
FOR ALL POSITIVE reals y.
As a last example, I will look
at something more interesting.
Consider the function f defined by:
f(x) = x².
Here is its graph.
We can’t quite find an inverse for it.
This function is NOT one-to-one.
For example, f(3) = f(-3). Therefore,
it does NOT have an inverse function.
However, we can still try to construct
something CLOSE to an inverse.
Here is the graph again.
Instead of the full function,
I could RESTRICT to a smaller domain
and consider only a “piece” of it.
For example,
I could keep just the function
represented by this piece of the graph,
in RED.
Then, I am maintaining the FULL range,
with a SMALLER domain,
and now the new function IS one-to-one,
so it will have an inverse!
To be clear, this is not
the only choice I could have made.
I could have chosen this
other half of the graph instead.
Or, if I get creative, perhaps this:
but that would be silly.
Instead,
let’s stick to my original choice.
What is the inverse...
...of the function represented by
the piece of the graph in red?
It is the square root!
Indeed,
let us call g the function that is
defined by the equation g(x) = √x.
g is NOT the inverse of the function f.
g is the inverse of the restriction of
the function f to the domain [0,∞).
This is actually how you learned of
square roots back in primary school,
even though you did not use these words.
For example,
you learned that there are two real
numbers whose square is 9: 3 and -3,
but that when we write √9, we mean
the positive "3".
So yes, the square root is the inverse of
the RESTRICTION of f to a smaller domain.
Equivalently, saying that a number "a",
is the square root of a number "b",
means that b is a²...
(if I were to stop here,
it would mean that f and g
were inverses of each other,
which they are not)
...and also that "a" is positive or zero.
But there is more!
We know that when we compose a
function and its inverse,
we should get the identity.
This will still be true,
as long as I stay in the right domain.
What is the √(x²)?
It is x, but only if x was positive
or zero to begin with;
otherwise, as you know,
it is the absolute value of x (|x|).
The moral is, if we have
a function that is NOT one-to-one,
we can restrict it to a smaller domain
and CONSTRUCT its inverse.
But the way we define inverse,
and the properties of inverse functions,
will have to be modified accordingly.
This will become very important when we
define inverse trigonometric functions.
